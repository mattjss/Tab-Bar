# Tab Bar — Morph Design

This is the **morph design** only: the middle tab bar frame expands in place (no flyout). When you click the hamburger, the same pill grows and the menu appears **above** the tab row inside that frame.

**Run the new design:**

```bash
cd tab-bar-morph
npm install
npm run dev
```

Then open **http://localhost:5174** in your browser.

- Port **5174** so it doesn’t conflict with the original app (usually 5173).
- Page title: **Tab Bar — Morph Design**.
